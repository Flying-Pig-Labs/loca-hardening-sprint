üéØ The Power of Context Files

The .context file is your project's knowledge base. It tells PromptDoctor everything about your application:

  • Technology stack and frameworks
  • Business rules and constraints
  • API structure and database schema
  • Security requirements and compliance needs
  • Performance targets and monitoring setup
  • Known issues and workarounds

With this context, PromptDoctor transforms vague requests like "add user login" into comprehensive, multi-step workflows tailored to YOUR specific application.

üîÑ How It Works

When you enter a request like "add shopping cart to the website", PromptDoctor:

  1. Analyzes your request to understand intent, scope, and risk
  2. References your .context file for project-specific details
  3. Generates research questions to uncover requirements
  4. Creates a planning phase with architecture decisions
  5. Provides validation steps before implementation
  6. Breaks down implementation into safe, atomic steps
  7. Includes comprehensive testing and deployment guidance
